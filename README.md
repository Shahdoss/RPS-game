# Rock, Paper, Scissors GAME

This is a browser-based Rock, Paper, Scissors game where the player must defeat an AI named ROBO-333. The project was developed to practice JavaScript fundamentals, including loops, functions, and conditional logic.

## How to Play
1. Open the Live Demo in your browser.
2. Open the Console.
3. Follow the instructions in the prompt windows.
4. Complete 5 rounds to see the final result.

## Technical Features
- Interactive gameplay: Uses prompt and alert for player interaction.
- Theme: An engaging backstory with dynamic console logging.
- Input validation
- Score tracking: Maintains a score for both the player and the computer throughout the game.

## Technologies Used
- HTML5
- JavaScript

## Project Requirements
- Implemented a function for random computer moves.
- Implemented a playRound function that returns the winner of each round.
- Created a game function with a 5-round loop and final score reporting.
- Focused on clean code, modular functions, and user experience.
